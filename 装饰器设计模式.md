### 装饰器设计模式

装饰器作为一个函数，接收函数作为参数，并在内部定义了一个函数，在这个函数内部中实现一些功能，并调用了作为参数传递进来的函数，最后将内部函数作为返回值。

函数的几个特性：

- 可以赋值给其他变量
- 可以作为参数传递给其他函数
- 可以在内部定义一个函数
- 可以当做返回值返回

##### 简单的装饰器（添加函数计时功能）	

```
import time

def fun(f):
    def statistics_time():
        start_time = time.time()
        f()
        end_time = time.time()
        print("do() time:{}".format(end_time-start_time))
    return statistics_time
# 方式一
def do():
    time.sleep(1)
# 方式二    
@fun
def do2():
	time.sleep(1)
if __name__ == '__main__':
	# 方式一
    do = fun(do)
    do()
    # 方式二
    do2()
```

##### 被装饰的函数带返回值

```
def fun(f):
    def statistics_time():
        start_time = time.time()
        data = f()
        end_time = time.time()
        print("do() time:{}".format(end_time-start_time))
        return data
    return statistics_time
    
@fun
def do2():
	time.sleep(1)
	return 1
if __name__ == '__main__':
    # 方式二
    data = do2()
    print(data)
```

##### 被装饰的函数带参数

```
def fun(f):
    def statistics_time(name):
        start_time = time.time()
        data = f(name)
        end_time = time.time()
        print("do() time:{}".format(end_time-start_time))
        return data
    return statistics_time
    
@fun
def do2(name):
	time.sleep(1)
	return name
if __name__ == '__main__':
    # 方式二
    data = do2('小明')
    print(data)
```

##### 装饰器本身带参数

使用装饰器时，会首先运行带参数的最外层函数，返回装饰器。

```
def add(id):
	def fun(f):
    	def statistics_time(name):
    		print("my id is :{}".format(id))
        	start_time = time.time()
        	data = f(name)
        	end_time = time.time()
        	print("do() time:{}".format(end_time-start_time))
        	return data
    	return statistics_time
    return fun
```

##### 多层装饰器

一个函数可以通过是被多个装饰器所装饰，执行顺序是从下到上的优先顺序加载装饰

```
@.....
@second
@first
def do():
	#
```

#### 类装饰器

类作为装饰器的时候，使用构造函数来传递参数（函数），并调用内部的_call_函数作为装饰器内部函数。

```
class A():
    def __init__(self,f):
        print("实例化一个A对象")
        self.f = f

    def __call__(self, *args, **kwargs):
        start_time = time.time()
        self.f()
        end_time = time.time()
        print("do函数运行时间：{}".format(end_time-start_time))

@A
def do():
    time.sleep(1)

if __name__ == '__main__':
    do()
```
